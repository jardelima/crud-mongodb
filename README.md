# CRUD com Express e MongoDB
CRUD de estudo feito com Express.js e MongoDB.

## O que utilizei
- Node
- Express
- Mongoose
- MongoDB Atlas
- Nodemon
- Postman

## Como rodar
Primeiro passo após clonar o repositório é instalar os pacotes:
`npm i`

E para rodar o projeto local:
`npm start`
